VSCode Keybindings

For advanced customizations, edit keybindings.json.

Default keybindings that tend to be very helpful:

Key Combination Result
alt + tab Changes application that currently has focus
ctrl + c Copies selected text to clipboard
ctrl + v Pastes text copied to clipboard
ctrl + x Cuts selected text to clipboard
ctrl + z Undo last command
ctrl + y Redo most recent undone; stores undone until a change is made
ctrl + tab Changes the tab that has focus in a tabbed application
ctrl + f Finds characters in the file that has focus**
ctrl + shift + f Finds characters in all files in the VSCode explorer**
**Use selectors beside the input field to check for case and exact match
ctrl + enter Creates a new line beneath the cursor and moves the cursor the new line
ctrl + 1/2/3 Moves focus between editor groups
ctrl + d Places an additional cursor on instances of selected text
ctrl + n Creates a new file
ctrl + left/right arrow key Moves cursor to respective end of current word
ctrl + shift + left/right Select word/part of word to the respective end of current word
alt + shift + f Format entire document
Chords Chords are two separate keypress actions.
ctrl + k, ctrl + f Format the selected block of text
ctrl + k, ctrl + o Open a folder
ctrl + k, z Toggles Zen mode
